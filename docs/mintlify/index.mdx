## What is GLuaLS?

GLua Language Server (GLuaLS) is a language server and toolkit for **Garry's Mod**, built on top of [EmmyLua Rust](https://github.com/EmmyLuaLs/emmylua-analyzer-rust).

It is built to work out of the box. Install the extension, open your GMod Lua project, and start coding. If you need custom settings, use the built-in settings panel.

## Core features

<CardGroup cols={2}>
  <Card title="Realm awareness" icon="globe" href="/language/realm-awareness">
    Detects client/server/shared realms from filenames, includes, and API usage. Warns about cross-realm mistakes in real time.
  </Card>
  <Card title="Network validation" icon="network-wired" href="/language/network-analysis">
    Validates `net.Start`/`net.Receive` payloads, catches mismatched write/read types, and delivers smart completion.
  </Card>
  <Card title="Scripted class support" icon="cube" href="/language/scripted-classes">
    Automatic detection of ENT, SWEP, TOOL, and custom classes. Synthesizes NetworkVar getters/setters and AccessorFuncs automatically.
  </Card>
  <Card title="Hook intelligence" icon="link" href="/language/hook-intelligence">
    Intelligent autocomplete for `hook.Add`, `hook.Run`, and `GM:` overrides. Registers custom hooks automatically.
  </Card>
  <Card title="Integrated debugger" icon="bug" href="/debugger/overview">
    Full SRCDS debugger with breakpoints, variable inspection, live entity explorer, and Lua execution straight from the editor.
  </Card>
  <Card title="AI integration" icon="wand-magic-sparkles" href="/vscode/ai-integration">
    Use Copilot and the built-in MCP tools to run Lua, read console output, and fetch server errors.
  </Card>
</CardGroup>

## Example: what GLuaLS catches

GLuaLS catches GMod-specific mistakes that generic Lua tools miss entirely:

<CodeGroup>
```lua Realm mismatch
-- sv_example.lua (server-side file)

-- Warning: draw.SimpleText is a clientside-only function
draw.SimpleText("Hello", "DermaDefault", 0, 0, Color(255, 255, 255))
```

```lua Network type mismatch
-- sv_send.lua
net.Start("PlayerData")
net.WriteString(ply:Nick())
net.WriteInt(ply:Frags(), 32)
net.Send(ply)

-- cl_receive.lua
net.Receive("PlayerData", function()
    local name = net.ReadString()
    local frags = net.ReadFloat() -- Warning: expected ReadInt, not ReadFloat
end)
```

```lua Invalid hook name
-- Warning: "PlayerSpawne" is not a known GMod hook name
hook.Add("PlayerSpawne", "MyAddon", function(ply) end)
```
</CodeGroup>
